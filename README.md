# Multiple Disease Prediction Web App

This is a machine learning-based web application that can predict the presence of multiple diseases, including:

- Diabetes  
- Heart Disease  
- Parkinson's Disease  
- Liver Conditions  
- Hepatitis  
- Jaundice  

The application allows users to enter relevant medical details, symptoms, and test results to get predictions based on trained ML models.
